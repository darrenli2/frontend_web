# README
Welcome to the Frontend_web repo. This repo is to build a frontend web application to connect the backend_api for user authentication.

## Environment
* `node >=5.7.0`
* `npm >=3.6.0`

## Running the application
* `$ npm install`
* `$ npm start`
*  Go to http://localhost:8000

## Running Tests
* `$ npm test`
